# find_cup_general 说明
---
## 1.使用方法

### 1.1 安装
```
cd catkin_ws
catkin_make
``` 

### 1.2 运行 

#### 测试USB相机 

```
roslaunch find_cup_ros find_cup_usbcam_general.launch
```
